Почта
==========
Методы для работы с почтой игра. Все методы требуют OAuth токен.

| Метод | Описание | OA2 |
| ----- | -------- |:---:|
| [GET /mail/](mail.md) | Получить почту игрока. | + |
| [PUT /mail/send/](mail.md) | Отправить письмо. | + |
| [DELETE /mail/delete/](mail.md) | Удаляет почту игрока. | + |
